Politique de confidentialité — ADDHAIBY

Dernière mise à jour : 08 juin 2026

Bienvenue sur ADDHAIBY — In Gold We Trust. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons vos informations lorsque vous utilisez notre application mobile.

1. Informations collectées

Notre application collecte les informations suivantes :

  • Données d'utilisation : pages visitées, actions effectuées dans l'app.
  • Token FCM : identifiant anonyme pour l'envoi de notifications push.
  • Préférences : langue choisie, mode sombre/clair (stockées localement).

Nous ne collectons pas de données personnelles identifiables (nom, adresse, numéro de téléphone) pour les utilisateurs standards.

2. Utilisation des données

Les données collectées sont utilisées pour :

  • Afficher les prix de l'or et de l'argent en temps réel.
  • Envoyer des notifications push lors des mises à jour de prix.
  • Améliorer l'expérience utilisateur.

3. Notifications Push

L'application utilise Firebase Cloud Messaging (FCM) pour envoyer des notifications concernant les prix de l'or et de l'argent. Vous pouvez désactiver ces notifications à tout moment depuis les paramètres de votre appareil.

4. Firebase et services tiers

Nous utilisons les services Google Firebase pour :

  • Firestore : stockage des prix et de l'historique.
  • Firebase Authentication : connexion sécurisée pour l'administrateur.
  • Firebase Messaging : envoi de notifications push.

Ces services sont soumis à la politique de confidentialité de Google.

5. Sécurité des données

Nous mettons en œuvre des mesures de sécurité appropriées pour protéger vos données contre tout accès non autorisé. Les données sont stockées sur les serveurs sécurisés de Google Firebase.

6. Droits des utilisateurs

Vous avez le droit de :

  • Désactiver les notifications push depuis les paramètres de votre appareil.
  • Supprimer les préférences locales en désinstallant l'application.
  • Nous contacter pour toute question concernant vos données.

7. Modifications de cette politique

Nous nous réservons le droit de modifier cette politique de confidentialité à tout moment. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.
